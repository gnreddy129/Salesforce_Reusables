Teams Webhook Configuration:
1. Go to your Teams channel
2. Click on "..." (more options) next to the channel name
3. Select "Connectors"
4. Find "Incoming Webhook" and click "Configure"
5. Give it a name like "Salesforce Test Results"
6. Copy the webhook URL and replace "YOUR_TEAMS_WEBHOOK_URL_HERE" in config.json

Report Portal Configuration (Optional):
1. Set up Report Portal instance
2. Get your API key from User Profile
3. Replace the placeholder values in config.json

Test Results File:
- The test-results.xml file will be generated automatically after test execution
- This file contains the test execution results in TestNG format
